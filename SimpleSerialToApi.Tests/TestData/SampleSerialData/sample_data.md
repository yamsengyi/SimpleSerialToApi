```
DEVICE:SENSOR001;TEMP:23.5C;HUMID:65.2%;PRESS:1013.25Pa;VOLT:12.4V;TIME:2023-01-01T12:30:45
DEVICE:SENSOR002;TEMP:24.1C;HUMID:63.8%;PRESS:1014.12Pa;VOLT:12.3V;TIME:2023-01-01T12:31:45

```
